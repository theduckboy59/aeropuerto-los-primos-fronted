# aeropuerto-los-primos-fronted

Frontend - Proyecto final Programacion 3

Descripción integral: Conecta el frontend con el backend Aeropuerto Los Primos  
Tecnologías: Angular 21.1.4, TypeScript, RxJS  
Funcionalidades alineadas: Vuelos, pasajeros, reservas, abordaje, reportes  
Guía de configuración: Integración con la API REST del backend  
Estructura del proyecto: Disposición de carpetas y componentes  
Instrucciones de desarrollo: Instalación, servidor local, construcción y pruebas
